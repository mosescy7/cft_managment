# CFT Apartments & Shops

Static website for **Cyabukombe Family Trading (CFT)** — furnished apartments and commercial shops for rent in Kimironko, Kigali, Rwanda.

## Pages

| File | Description |
|---|---|
| `index.html` | Home page |
| `apartments.html` | Apartment listings overview |
| `apartment-kimironko.html` | Kimironko apartment detail page |
| `apartment-vision-city.html` | Vision City apartment detail page |
| `shops.html` | Commercial shop listings |
| `gallery.html` | Photo gallery |
| `about.html` | About the company |
| `contact.html` | Contact information |
| `booking.html` | Booking page |

## Project structure

```
css/        Stylesheet (styles.css)
js/         Site behavior (main.js)
images/     Property photos
build.py    Python script that generates the HTML pages from a shared header/footer shell
```

## Development

The pages share a common header/footer/nav structure generated by `build.py`. To regenerate the HTML after editing shared layout, contact info, or links, update the constants and templates in `build.py` and re-run it:

```bash
python build.py
```

Before publishing, replace the placeholder contact details in `build.py` (WhatsApp number, phone, email).

## Deployment

This is a static site — any static host (GitHub Pages, Netlify, Vercel, etc.) can serve it directly from the repository root.
